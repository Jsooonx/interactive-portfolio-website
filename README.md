# Interactive Portfolio Website

A personal portfolio website showcasing programming projects and technical learning progress.  
Built using HTML, CSS, and vanilla JavaScript with a focus on responsive design and clean UI structure.

## Features
- Responsive layout for desktop and mobile
- Project showcase section
- Interactive navigation and animations
- FAQ and contact sections

## Concepts Practiced
- Semantic HTML structure
- Responsive web design
- JavaScript interactivity
- UI component organization

## Technologies
- HTML
- CSS
- JavaScript
